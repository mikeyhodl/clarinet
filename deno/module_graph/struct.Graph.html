Given a string specifier and a referring module specifier, provide the resulting module specifier and media type for the module that is part of the graph.

Arguments

  • specifier - The string form of the module specifier that needs to be resolved.
  • referrer - The referring ModuleSpecifier.
  • prefer_types - When resolving to a module specifier, determine if a type dependency is preferred over a code dependency. This is set to true when resolving module names for tsc as it needs the type dependency over the code, while other consumers do not handle type only dependencies.
